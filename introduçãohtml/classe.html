[ Classe ]

para conseguri selecionar a classe no css atribuimos um " . " depois o nome da classe

No contexto do HTML, a palavra "classe" refere-se a um atributo utilizado para definir um nome único ou conjunto de nomes que podem ser aplicados a elementos HTML. Esses nomes de classe são usados principalmente para aplicar estilos CSS e para seleção e manipulação através de JavaScript.


Estilização CSS: Classes são frequentemente usadas para aplicar estilos específicos a elementos. Por exemplo, você pode definir regras CSS para uma classe chamada "destaque" que faz com que o texto seja exibido em negrito e com uma cor de fundo amarela. Dessa forma, você pode aplicar esses estilos a vários elementos em diferentes partes do seu site.


Classes também são usadas para selecionar elementos em JavaScript. Isso permite que você acesse e manipule os elementos de maneira mais direcionada. Classes também podem ser usadas para definir comportamentos específicos em JavaScript. Por exemplo, você pode adicionar uma classe "ativo" a um elemento para indicar que ele está ativo, e então usar essa classe para controlar comportamentos específicos. Lembre-se de que uma mesma classe pode ser aplicada a vários elementos diferentes, permitindo que você reutilize estilos e comportamentos semelhantes em todo o seu site. No geral, as classes no HTML são uma maneira poderosa de organizar, estilizar e interagir com elementos de forma mais eficiente e estruturada.
