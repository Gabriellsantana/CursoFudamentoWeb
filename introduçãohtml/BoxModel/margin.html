margin - MARGEM

blocom

  • "margin-top": Define a margem superior do elemento.
  • "margin-right": Define a margem direita do elemento.
  • margin-bottom": Define a margem inferior do elemento.
  • margin-left": Define a margem esquerda do elemento

O espaçamento definido pelo "margin" pode ser aplicado externamente, empurrando os elementos vizinhos para longe, ou internamente, afetando o espaço dentro do elemento. Você pode usar valores negativos no "margin" para criar sobreposições entre elementos ou ajustar o espaçamento com mais precisão.
